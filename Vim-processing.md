###### vim常用指令合集

---



> **1. 当前窗口打开多个文件：
>
> ```sh
> :sp file //水平分割打开第N个文件
> ```
>
> ```sh
> :vsp file //垂直分割打开第N个文件
> ```
>
> `使用ctrl + ww可以在各个文件之间切换`
